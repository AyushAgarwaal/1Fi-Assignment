# 1FI SDE-1 Assignment

## Setup

### Backend

```bash
cd backend
npm init
npm install
=>SET .env file:
    MONGODB_URI= Your database URL
    PORT=5000

=>SET package.json:
    "scripts":
        "start": "node index.js",
        "dev": "nodemon index.js"

npm run dev
```
### Frontend

```bash
cd frontend

npm start
```
